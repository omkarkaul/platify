About

Why'd you build this?
Cos I felt like it
So that I could practice building a reverse proxy and doing static rendering in node, also so I would have a number plate lookup tool that had the functionality of Kiwiplates and Carjam combined. Obviously a lot of functionality still needs to be implemented, and I have a trillion ideas about how I could make this really interesting, but it solves the problem I was having for now

How does it work?
The front end fetches plate data via a reverse proxy. The proxy fetches plate data from some publicly exposed API's, does some fiddling, and then returns it to the front end. We also have the luxury of being able to lookup confidential plates that NZTA requests blocked access to from conventional number plate lookups via services like Carjam
